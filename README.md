# Name

SSSM ssBigClock Version002

## Overview

Windows用スクリーンセーバー(多分7以降なら動くのでは？Win10で開発)
画面いっぱいの大きなデジタル時計
マルチモニタ対応
フォント指定
表示色指定
背景色指定

## Requirement

Windows7以降?
開発と動作確認はWindows10
Delphi XE10.3で開発

## Setup

ssbigclock.scr を C:\Windows\System32 へコピー
